# AutoLayoutGallery
Typecho文章页相册插件
为简约默认主题，适配fancybox灯箱图库
已经适配的主题会出现冲突
使用短代码插入
“
[photos]
![图片.jpg](图片.网址)
![图片.jpg](图片.网址)
[/photos]
”
